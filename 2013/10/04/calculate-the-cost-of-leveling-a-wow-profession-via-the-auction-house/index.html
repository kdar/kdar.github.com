Calculate the cost of leveling a WoW profession via the auction house

1 min

ProfessionCost

Determines how much a WoW profession would cost if you bought everything from the AH.

Download

ProfessionCost.zip

Usage

Go to https://theunderminejournal.com/user.php and download the Market Data XML file for your particular realm. Put this file into the data directory. Then, just run professioncost.exe.
